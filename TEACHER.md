# 가이드(선생님) 기능 요청

> 선생님 입장에서 필요한 기능, 개선 아이디어, 미구현 요청을 적는 파일.
> 영준/레제가 이 파일 보고 개발 우선순위 결정.

---

## 현재 구현된 기능

| 기능 | 상태 | 경로 |
|------|------|------|
| 퀘스트 생성/관리 | ✅ 완료 | `/teacher/missions` |
| 빌더 목록 / 코인 관리 | ✅ 완료 | `/teacher/classes` |
| 자리배치도 | ✅ 완료 | `/teacher/tools` |
| QR코드 초대 | ✅ 완료 | `/teacher/qrcodes` |
| 가이드 가이드 | ✅ 완료 | `/teacher/guide` |

---

## 기능 요청 (미구현)

> 필요한 기능 생기면 여기 추가해줘. 우선순위(높음/중간/낮음)도 같이 적어주면 좋음.

| 기능 | 설명 | 우선순위 | 요청자 |
|------|------|----------|--------|
| 알림장 | 하루 공지/숙제 공지를 빌더들에게 전송 | - | - |
