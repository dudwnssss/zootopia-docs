# 가이드(선생님) 기능 요청

> 선생님 입장에서 필요한 기능, 개선 아이디어를 적는 파일.
> 요청 → 레제가 영준 컨펌 → 영준 OK → 개발 시작 순서로 진행됨.

---

## 처리 프로세스

| 변경 종류 | 처리 방식 |
|----------|----------|
| 세계관/용어/텍스트 변경 | 레제 바로 반영 |
| 기존 기능 UI 개선 | 레제 바로 반영 |
| **새 기능 추가** | **영준 컨펌 후 진행** |

---

## 현재 구현된 기능

| 기능 | 상태 | 경로 |
|------|------|------|
| 퀘스트 생성/관리 | ✅ 완료 | `/teacher/missions` |
| 빌더 목록 / 코인 관리 | ✅ 완료 | `/teacher/classes` |
| 자리배치도 | ✅ 완료 | `/teacher/tools` |
| QR코드 초대 | ✅ 완료 | `/teacher/qrcodes` |
| 가이드 가이드 | ✅ 완료 | `/teacher/guide` |

## 기능 요청

> 새 기능 아이디어 생기면 여기 추가해줘. 레제가 영준한테 전달하고 컨펌 받아서 진행할게.

| 기능 | 설명 | 상태 | 비고 |
|------|------|------|------|
| 기본 알림장 | 가이드가 알림장 작성, 반별 목록, 학생 홈에서 열람 | ✅ 완료 | `/teacher/notices` |
| AI 알림장 | AI 자동작성 (안전지도/생활지도 멘트, 숙제 자동 기입) | ⏸ 보류 | 프로 버전, 추후 논의 |
| 자리배치 템플릿 저장 | 선호 자리배치 저장 (기본 3개, 프로 무제한) | ⏳ 컨펌 대기 | 2026-02-25 준엽 요청 |
| 화폐 순차적 해방 | 일정 레벨 이상 빌더부터 가챠 외 상품 구매 가능 | ⏳ 컨펌 대기 | 학기초엔 상품 없어서 잠근다는 컨셉. 2026-02-25 준엽 요청 |
| 주목 효과음 | 음성파일 업로드 또는 AI 템플릿 제작으로 주목 효과음 설정 | ⏳ 컨펌 대기 | 2026-02-25 준엽 요청 |
| 미션 DB | 미션 데이터 DB화 (빌더 목록에서 미션 참조 구조) | ⏳ 컨펌 대기 | 2026-02-25 준엽 요청, 구체 범위 확인 필요 |
| 동물 DB | 동물 캐릭터 DB화 | ⏳ 컨펌 대기 | 2026-02-25 준엽 요청, 구체 범위 확인 필요 |
| 개인농장 꾸미기 | 가챠 동물로 개인 농장 공간 꾸미기 | ⏳ 컨펌 대기 | 배포 전 완성 목표. 2026-02-25 준엽 요청 |
| 마을(학급 미니홈피) 꾸미기 | 학급 단위 공간 꾸미기 | ⏳ 컨펌 대기 | 배포 전 완성 목표. 2026-02-25 준엽 요청 |

**상태값**: ⏳ 컨펌 대기 / 🔄 진행 중 / ✅ 완료 / ⏸ 보류

## 현장 인사이트

> 선생님 니즈, 사용 패턴 등 개발 참고용 메모.

- **일방소통 선호**: 선생님들은 개인적 연락 없이 공지만 하고 싶어함. 전화는 교실 내선만 허용하는 분위기. → 알림장/공지 기능이 핵심
- **배포 전략** (2026-02-25 준엽): 기본 학급관리+안내 시스템 완성 + 개인농장 꾸미기 + 학급 마을 꾸미기 세트 완성 후 배포 시작
