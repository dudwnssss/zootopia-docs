# 가이드(선생님) 기능 요청

> 선생님 입장에서 필요한 기능, 개선 아이디어를 적는 파일.
> 요청 → 레제가 영준 컨펌 → 영준 OK → 개발 시작 순서로 진행됨.

---

## 처리 프로세스

| 변경 종류 | 처리 방식 |
|----------|----------|
| 세계관/용어/텍스트 변경 | 레제 바로 반영 |
| 기존 기능 UI 개선 | 레제 바로 반영 |
| **새 기능 추가** | **영준 컨펌 후 진행** |

---

## 현재 구현된 기능

| 기능 | 상태 | 경로 |
|------|------|------|
| 퀘스트 생성/관리 | ✅ 완료 | `/teacher/missions` |
| 빌더 목록 / 코인 관리 | ✅ 완료 | `/teacher/classes` |
| 자리배치도 | ✅ 완료 | `/teacher/tools` |
| QR코드 초대 | ✅ 완료 | `/teacher/qrcodes` |
| 가이드 가이드 | ✅ 완료 | `/teacher/guide` |

**상태값**: ⏳ 컨펌 대기 / 🔄 진행 중 / ✅ 완료 / ⏸ 보류
