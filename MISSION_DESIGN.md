# Zootopia 미션 설계 가이드

## 핵심 철학
**"주민이 내주는 미션 = 그 주민의 삶에서 나온 것"**

미션은 학습 목표를 직접 드러내지 않는다.
부기(부엉이 선생님)가 내주는 퀘스트를 완료하면 자연스럽게 학습이 된다.
아이들은 "공부"가 아니라 "부기를 도왔다"고 느낀다.

## 친밀도 시스템 세계관

> "미션을 완료할수록 그 주민과 가까워져. 친밀도가 높아지면 주민이 마을에서 더 활발하게 활동해."

| 친밀도 | 상태 | 효과 (로드맵) |
|--------|------|---------------|
| 0 | 낯선 사이 | - |
| 1~10 | 알고 있는 사이 | 캐릭터 대사 해금 |
| 11~25 | 친한 사이 | 뽑기 확률 소폭 UP |
| 26~49 | 절친 | 전용 대사 + 특별 이펙트 |
| 50 | 마을 베스트 프렌드 💝 | 전용 칭호 + 보너스 뽑기 티켓 |

## 미션 설계 원칙 (선생님용)

1. **주민의 직업에서 미션이 나온다** — "부기가 퀴즈를 낸다"는 자연스럽다
2. **학습 목표를 숨겨라** — "수학 공부"가 아니라 "담이의 집 설계 도와주기"
3. **즉시완료 미션** → 빠른 성취감 (일상 습관)
4. **선생님 확인 미션** → 참여 확인이 필요한 것 (발표, 실험)
5. **퀴즈 미션** → 지식 확인 (객관식 or OX)
6. **전설 주민 미션** → 특별한 날, 이벤트, 학기 말 등에만
